# Cybersecurity Foundations

## Overview
This repository contains my structured learning journey in cybersecurity starting from Pre-Security level. It includes notes and summaries from TryHackMe and other foundational cybersecurity topics.

The focus is on building strong basics in Linux, networking, web fundamentals, and core security concepts.


## Learning Goals
- Understand how the internet and networks work
- Learn Linux command-line basics
- Understand how websites and servers communicate
- Build awareness of basic cybersecurity concepts and threats
- Develop structured technical note-taking habits



## Purpose
This repository is for personal learning and skill development in cybersecurity fundamentals. It also serves as a structured record of my understanding from hands-on labs and practice.



## Disclaimer
All content is created for educational and ethical cybersecurity learning purposes only.
